# with jb's react-boilerplate

> https://github.com/jb-/jb-react-boilerplate

## Getting started

To install dependencies; `$ yarn` 

And then, to run the devserver; `$ yarn start` 

or to bundle for the production; `$ yarn build`

## List of packages

* react
    * react-dom
* babel
    * babel-core
    * babel-cli
    * babel-preset-env
    * babel-preset-react
* webpack
    * webpack-cli
    * webpack-dev-server
    * html-webpack-plugin
